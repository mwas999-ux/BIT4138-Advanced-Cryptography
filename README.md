# Secure Login & Authentication System

A cryptography-based secure login system built for BIT4138: Advanced Cryptography.

## Tech Stack
- Python 3.14
- Flask 3.1.3
- Python Cryptography Library
- OpenSSL 3.5.3
- MySQL / SQLite
- Flask-Bcrypt
